# How do i get the mod ?
1. Install Minecraft 1.7.10
2. Install Forge for 1.7.10
3. Go to 'Releases' and download the newest version
4. Throw the .jar in your 'mods' folder
5. Start up the game
6. HAVE FUN WITH SHINY THINGS :D
